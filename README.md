## üöÄ Deployment & Access Guide

### 1. Prerequisite Infrastructure
Ensure your environment meets the following specifications:
*   **Node.js**: Version 18.0 or higher
*   **Python**: Version 3.9 or higher

### 2. Implementation

#### **A. Frontend HUD**
```bash
# Clone the intelligence repository
git clone https://github.com/Aaryankansari/atlas-canvas.git
cd atlas-canvas

# Initialize dependencies
npm install

# Launch the Tactical Workspace
npm run dev
```

#### **B. OSINT Researcher Backend**
```bash
# Navigate to the engine core
cd backend

# Establish virtual environment
python -m venv venv
source venv/bin/activate # or venv\Scripts\activate on Windows

# Install researcher requirements
pip install -r requirements.txt

# Start the Intelligence Engine
python main.py
```

### üß† Level-3 AI Configuration (Gemini)
To enable real-time, LLM-driven intelligence synthesis:
1. Create a `.env` file in the `/backend` directory.
2. Define your Google Gemini API Key: `GEMINI_API_KEY=your_secured_key`.
3. Restart the engine. Atlas will automatically pivot from mock-bio generation to Gemini-powered deep research.
